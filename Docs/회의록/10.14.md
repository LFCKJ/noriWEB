### 주요 기능

#### 일정

-   **정의**: 사용자가 정의한 일정을 저장 및 처리하는 프로그램의 주기능
-   **상세**: CRUD

##### 일정 타입

-   **캘린더**
    -   내용: 기본적인 알림 등록
-   **카운터**
    -   내용: 매 주기마다 리셋되는 카운터 일정
    -   예시: 매일 물 마시기 8회
-   **프로그레스**

    -   내용: 기준을 둬서 해당 목표치에서 현재치의 비율
    -   예시: 100시간 공부하기, 개발 진척 등

    -   타입
        -   수치
        -   시간

-   **테스크**

    -   내용: Todo
    -   예시: 양파 사오기, 보고서 작성하기

#### 태그

-   일정 등록시 태그 속성을 포함해서 이후에 필터 등에서 사용할 수 있게 함
-   태그 이름, 색 저장

#### 알림

-   **정의**: 등록한 일정이나 사용자 설정에 따라서 사용자에게 알림을 보내는 기능

-   **상세**:
    -   회원 가입
    -   로그인, 로그아웃
    -   회원 탈퇴
    -   프로필 등록, 수정

#### 분석

-   **정의**: 사용자의 일정관리습관을 추적하고 분석하여 피드백을 보여주는 기능
-   **상세**:

    -   시각화
