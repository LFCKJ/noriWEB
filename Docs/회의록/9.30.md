## 9.30 회의록

9.30 회의사항

### 웹페이지 디자인 건

아직 구체적인 디자인은 나오지 않았으나 대략적인 틀을 잡음

-   메인페이지
    -   노션의 웹 페이지를 참고하여 구성
    -   로그인 버튼을 누르면 로그인 다이얼 로그가 뜨도록 구현
-   대시보드 페이지
    -   사이드 바를 활용
        -   아래 항목이 포함되어야 함
            -   계정 전환
            -   각 계정에 대한 탭
            -   설정
        -   사이드 바는 접히도록 구현
            -   노션의 사이드바 참고
    -   기본 화면은 사용자 구성 페이지로 정함
    -   기본 화면의 첫 페이지는 템플릿 중에서 선택하여 구성할 수 있도록 제공
        -   노션의 페이지 구성과 유사하게 구현
        -   이후 사용자가 직접 페이지를 구성할 수 있도록 제공
    -   오른쪽 하단에 + 버튼을 두어, 사용자가 요소를 구성할 수 있도록 함
